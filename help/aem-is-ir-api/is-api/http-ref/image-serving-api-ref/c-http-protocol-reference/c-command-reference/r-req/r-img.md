# img{#img}

Bild (standard). Begär standardbilddata.

`req=img`

Svarsdataformatet och MIME-typen för svar bestäms av `fmt=`. `req=img` är standardbegärandetypen och behöver inte anges explicit. HTTP-svaret kan nås med TTL-värdet baserat på `catalog::Expiration`.

Andra begärandekommandon används som dokumenterat.

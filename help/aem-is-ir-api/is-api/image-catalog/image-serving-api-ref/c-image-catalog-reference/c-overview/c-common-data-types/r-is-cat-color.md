# Färg{#color}

Färgdatatyp.

Färgvärde. Hexadecimalt, packat RGB-värde, eventuellt föregånget av `0x`. RGB-värdet `128,255,0` kan till exempel anges som `0x80ff00` eller `80ff00` .

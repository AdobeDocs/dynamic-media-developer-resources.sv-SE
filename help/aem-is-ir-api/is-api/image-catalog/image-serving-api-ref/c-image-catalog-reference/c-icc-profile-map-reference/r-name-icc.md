# Namn{#name}

Namn på genväg till profil. Ett kortkommandonamn för färgprofilen.

## Egenskaper {#section-30fc50f71f4a4ea7b14b2524d1f2b698}

Textsträng. Obligatoriskt. Primär indexnyckel för ICC-profilmappningen. Varje `icc::Name`-värde måste vara unikt i tabellen och får inte innehålla &#39;,&#39;-tecken.

## Standard {#section-47a56671d1e94497aa7ea83fe9590c05}

Ingen.

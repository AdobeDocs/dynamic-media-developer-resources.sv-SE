# Illum{#illum}

Väljare för illustrationsschema. Det gör att det går att uttryckligen välja belysningskartan när materialet återges.

## Egenskaper {#section-162bcf562ca844ccba9e81e267508cca}

Enum. Ange -1 för automatiskt val av belysningskarta baserat på värdet för katalogen::Gloss.

Ange 0, 1 eller 2 för att välja belysningskarta A, B eller C. Återgivaren väljer det närmaste belysningskarta som finns i vinjetteringen.

## Standard {#section-ac386d31ef90423b8a367010a60bddc7}

-1 (automatisk markering)

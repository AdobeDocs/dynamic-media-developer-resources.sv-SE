# Typ{#type}

Materialtyp. Materialets yttyp. Används tillsammans med katalog::Glans och katalog::Grovhet för att styra återgivningseffekter för 3D-reflektion.

## Egenskaper {#section-86e8bc194f764c848e0ee55630a5ac1b}

Enum. Valfritt för alla material. Ignoreras om vinjetteringen inte har någon återgivningsfunktion för 3D-reflektion.

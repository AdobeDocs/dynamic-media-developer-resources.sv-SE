# Installation av Image Serving {#installation}

Installation av Image Serving innefattar två större serverkomponenter:

* Bildserver. En fristående, inbyggd kodkörbar fil.
* Plattformsserver. En fristående Java™-webbapplikation och applikationsserver.

Bildåtergivning är en del av dessa två serverkomponenter.

I följande instruktioner `install_root` refererar till rotmappen där Image Serving och Image Rendering är installerade. Och `server:port` refererar till värdnamnet eller IP-adressen och portnumret som konfigurerats som plattformsserverns huvudavlyssningsport.

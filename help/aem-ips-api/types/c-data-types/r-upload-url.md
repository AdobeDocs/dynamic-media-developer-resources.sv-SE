## Parametrar {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| Namn | Typ | Beskrivning |
|---|---|---|

| `*`destPath`*` | `xsd:string` | IPS-filens sökväg och namn för filen som du vill överföra till. |
